## Client side
### Tools
  - framework react
  - styles tailwind https://flowbite.com/docs/components
    - heroicons https://heroicons.dev/

### Run in development mode
First of all install the all dependencies with
```
$ npm install
```

After you can run the app with
```
$ npm run start
```
